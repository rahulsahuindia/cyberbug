# cyberbug# 🔥 CyberBug - Smart Recon & Bug Hunting Tool

CyberBug ek beginner friendly cyber security research automation tool hai  
jo attack surface mapping aur basic vulnerability hunting me help karta hai.

## 🚀 Features

- ✅ Target URL crawling  
- ✅ Hidden endpoints detection  
- ✅ JavaScript file extraction  
- ✅ Parameter discovery  
- ✅ Basic vulnerability hints  

## 🛠️ Installation

Clone the repository:

git clone https://github.com/YOUR_USERNAME/cyberbug.git

Go to folder:

cd cyberbug

Install requirements:

pip install -r requirements.txt

## ▶️ Usage

python tool.py

Then enter target URL.

## ⚠️ Disclaimer

This tool is made for **educational and authorized security testing purposes only.**  
Do not use this tool on targets without permission.
